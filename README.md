# Subvenia Platform

Plataforma SaaS para clusters, cámaras de comercio y asociaciones empresariales que permite automatizar la detección de subvenciones y ayudas para sus empresas asociadas.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `VITE_FUNCTION_SECRET_KEY` in [.env.local](.env.local) 
3. Run the app:
   `npm run dev`
